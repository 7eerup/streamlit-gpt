## Clone The Repository
```bash
git clone https://github.com/7eerup/streamlitGPT.git
```

## CD Into Directory
```
cd streamlitGPT
cd main
```

## Set up .venv
```bash
python -m venv .venv
source streamlitGPT/main/.venv/bin/activate
```

## Start Server
```bash
streamlit run app.py
```
